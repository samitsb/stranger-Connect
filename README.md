# stranger-Connect

Stranger Connect is a frontend web app built as a Bachelor project for safe global chatting.
The project focuses on creating an engaging UI for users to discover the service, sign up, log in, and access chat features.

## Pages

- `landing/landing.html` — landing page with the app introduction and entry points for login/signup.
- `login/login.html` — login page for returning users.
- `signup/signup.html` — signup page for new users.
- `chat/chat.html` — chat interface for messaging.
- `aboutUs/aboutUs.html` — about us page describing the project and team.

## Frontend structure

The client folder contains:
- `page/landing/` for the landing page UI
- `page/login/` for login functionality and styling
- `page/signup/` for registration forms and scripts
- `page/chat/` for the chat interface and its logic
- `page/aboutUs/` for project information and styling

Each page folder includes its own CSS and JavaScript files where needed.

## Current status

Open the desired HTML file from `client/page/` in a browser to preview the page.
